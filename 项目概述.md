# 不牛马厨房｜项目概述

> 用途：帮助 AI 快速理解项目。功能设计、产品结构、核心交互、业务规则、项目定位、架构或目录变化时必须同步更新。

## 项目介绍

“不牛马厨房”是一个手机优先的做饭决策网站，核心问题是“今晚吃什么”。用户可通过转盘抽菜，并结合人数、辣度、偏好、忌口、健康模式和现有食材筛选结果，随后查看做法、管理购物清单和记录最近做过的菜。

## 产品结构

- `首页`：展示抽菜结果、人数及筛选入口。
- `菜库`：搜索和分类浏览菜品，可选菜或加入清单。
- `食材`：冰箱食材匹配与购物清单。
- `我的`：角色/主题装扮、宠物主题联动、健康模式和添加自定义菜。
- `全屏转盘`：首次打开显示，也可从首页进入；包含转盘减速、宠物完成动画和结果回填。

## 技术与目录

- 形态：无构建工具的单文件静态网站，使用 HTML、内联 CSS 和原生 JavaScript。
- 主文件：`site/bu-niu-ma-kitchen.html`
- 根目录 `index.html`：GitHub Pages 与自定义域名入口，自动跳转到 `site/bu-niu-ma-kitchen.html`。
- 素材目录：`site/assets/`
- 项目协作文件：根目录 `AGENTS.md`、`项目概述.md`、`网站进度表.md` 均随 Git 仓库发布；AI 开始任务前必须先读取后两者，并遵循 `AGENTS.md` 的约束。
- 正式维护范围：仅当前工作区的 `site`；不要修改旧 `Codex/.../outputs` 路径。
- 页面最大宽度 620px，采用语义化主题色令牌和本地“乐米小奶泡体”；内置青柠绿、奶油杏、番茄红、莓果粉、海盐蓝、紫薯紫六套配色。
- 图标依赖 Lucide CDN；菜图仅来自原始作品页可核验的授权来源（当前为 Wikimedia Commons、Flickr 与 Pexels），本地化文件在 `site/assets/dishes/` 中保留作者、来源与许可；仅收录 CC0、公共领域、CC BY、CC BY-SA，或原始 Pexels 作品页标注“Free to use”且符合 Pexels License 的具体成品图。Pexels 图仅作为菜谱内容展示，不作为图库再分发，避开明显人物、商标和包装。发布菜图统一为最大边 640px、质量 76 的 WebP；加载顺序为当前转盘 8 个候选与抽中结果优先、菜库首屏 6 张立即加载、其余图片在接近视口时分批加载，避免首次进入同时请求整库图片。当前选中宠物的 `idle`、`runRight`、`runLeft`、`done` 四个动作在页面初始化或换宠物后立即高优先级预载，其他宠物只在接近宠物设置区时加载待机缩略图；动作重播复用浏览器缓存，不再追加时间戳请求。本地字体和角色素材可离线使用。

## 核心规则

- 每轮转盘最多 8 个候选；支持点击和拖拽抽菜。
- 支持 1、2、3、4+ 人，多人时可搭配多道菜。
- 优先避开最近 3 次做过的菜，无候选时回退。
- 忌口检查菜名、标签、食材、健康提醒和做法。
- 健康模式包括术后恢复期和低碘预案，仅作产品提醒与筛选，不作为医疗建议。
- 菜图只使用原始页可核验授权的具体成品图；已本地化的实拍图可离线展示并附作者、来源页和许可，未收录或网络加载失败时显示“暂无可授权实拍图”，不使用抽象图、无关通用食物图或破损图片。每道菜当前只绑定一张已核验图片，因此首页结果和菜库卡片不提供换图按钮，避免单图被切换为空。Pexels 图须在原始页标注“Free to use”，并且不使用明显人物、商标或包装。
- 内置菜 240 道：125 道种子菜 + 115 道扩展菜，另叠加用户自定义菜。
- 默认角色为草莓甜心小厨；宠物区同时保留奶茶小狗、Astr 和包仔小厨，所有角色统一包含 `idle`、`runRight`、`runLeft`、`done` 四种状态。
- 主题区保留六套互不重复的餐饮配色；旧版浅林绿、深森绿设置会自动迁移到青柠绿。
- 宠物主题与自由配色并存：宠物主题以全局色调统一品牌氛围，将动态图案主要集中在底部四栏导航的选中和点击反馈，避免正文卡片过度装饰；手动选择自由配色会退出联动。当前已完成“草莓甜心厨房”样板，其他宠物主题待后续接入。

## 本地数据

| localStorage 键 | 内容 |
|---|---|
| `bnmKitchen.customRecipes` | 自定义菜 |
| `bnmKitchen.history` | 最近做过 |
| `bnmKitchen.shopping` | 购物清单 |
| `bnmKitchen.settings` | 人数、筛选、健康模式、忌口、冰箱输入、自由主题及宠物主题联动状态等 |
| `bnmKitchen.mascot` | 当前角色 |
| `bnmKitchen.mascotRevision` | 角色资源迁移版本 |

## 运行与验收

- 双击根目录 `启动预览.cmd`，访问 `http://127.0.0.1:8787/bu-niu-ma-kitchen.html`。
- 出现 `ERR_CONNECTION_REFUSED` 或角色图片解码为 `0×0` 时，重新运行启动脚本并刷新。
- 修改前备份主 HTML；修改后按改动范围检查抽菜、连续重抽、筛选、菜库、冰箱推荐、清单、加菜、装扮及刷新后的本地数据。
